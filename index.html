FAQ

Preciso ter experiência prévia em programação? +

Não. O kit começa do zero absoluto — o Módulo 1 explica o que é JavaScript e como rodar seu primeiro código. Se você sabe usar um computador, já é o suficiente.

O kit é um PDF ou um curso em vídeo? +

É um livro digital em PDF — completo, com código, exercícios e gabarito. A vantagem é que você lê no seu ritmo, sem depender de plataforma de vídeo.

Qual sistema operacional é necessário? +

Qualquer um — Windows, Mac ou Linux. O kit ensina a instalar o VS Code e o Node.js nos três sistemas. Você precisa só de um computador com acesso à internet.

Vou conseguir criar projetos reais ao terminar? +

Sim. O Módulo 12 é dedicado aos seus primeiros projetos reais, e o kit termina com um roteiro claro do que estudar depois — React, Node, TypeScript — para continuar evoluindo.

O acesso expira? +

Não. É um pagamento único e o PDF é seu para sempre. Baixe e guarde onde quiser — sem plataforma de streaming, sem dependência de login.
